README
nic-cage.csv has a list of all Nic Cage acting roles from IMDb 
https://www.imdb.com/name/nm0000115/#actor
I chose the categories "movie", "character", "voice", and "year."
Movie and character are both strings with movie title and character name respectively
Voice is binary: 0 indicates live action and 1 indicates a voice acting role
Year is numerical
